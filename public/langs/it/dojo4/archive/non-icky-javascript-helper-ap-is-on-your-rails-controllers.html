Le web app finiscono quasi sempre per aver bisogno di una serie di piccoli metodi helper javascript: sai, moduli di autocompletamento, popolamento dei valori predefiniti, validazioni, ecc. Questi non sono chiamate API vere e proprie, solo piccoli frammenti di funzionalità necessari per far funzionare le viste.

C'è sempre la questione di quale controller associare questi metodi. Si crea un controller helper globale per tutto questo? Si associano al controller in questione? Come si condivide il backend dei metodi helper javascript tra i controller?

passo uno

Includere il modulo RPC nel ApplicationController

  class ApplicationController < ::ActionController::Base
    include RPC
  end

passo due

Nei controller, dichiarare i metodi helper rpc

  class PostsController < ApplicationController
    rpc(:defaults) do |params|
      post = Post.new(params)
      post.set_defaults!
      post.as_json
    end
  end

passo tre

Configura js per poter chiamare i metodi rpc relativi al controller corrente

  // in file: app/views/layouts/application.html.erb
  //
  // nota il *relativo* url endpoint rpc
  //
  // <script>
  
    jQuery(function(){
        var rpc = function(){
        // analizza gli argomenti
        //
          var args = Array.prototype.slice.call(arguments);
          var ajax;
          var method;
          if(args.length == 1 && typeof(args[0]) == 'object'){
            ajax = args.shift();
          } else {
            ajax = {};
            for(var i = 0; i < args.length; i++){
              var arg = args[i];
              switch (typeof(arg)) {
                case 'object':
                  ajax['data'] = arg;
                  break;
                case 'function':
                  ajax['success'] = arg;
                  break;
                case 'string':
                  method = arg;
                  break;
              };
            }
          };
        // normalizza le opzioni ajax
        //
          ajax['url'] = ajax['url'] || rpc['url'];
          method = method || ajax['method'];
          delete ajax['method'];
          ajax['data'] = ajax['data'] || {};
          ajax['data']['method'] = method;
          ajax['async']          = true;
          ajax['type']           = 'GET';
          ajax['dataType']       = 'json';
          ajax['cache']          = false;
          ajax['contentType']    = 'application/json; charset = utf-8';
          var result = ajax;
        // richiesta ajax - utilizzando callback o restituendo il risultato in stile sync
        //
          if(!ajax['success']){
            ajax['async'] = false;
            ajax['success'] = function(response){ result = response };
          };
          jQuery.ajax(ajax);
          return(result);
        };
        
        
        rpc['url'] = <%= raw url_for(:action => :rpc).to_json %>;
        
        
        window.rpc = rpc;
    });
  
  // </script>

passo quattro

Abilitare gli endpoint rpc in config/routes.rb

  resources :posts do
    collection do
      get 'rpc'
    end
  end
