Avec les déploiements Rails + Capistrano, ce paramètre

ssh_options[:forward_agent] = true

est pratiquement le défaut de facto et sain. Il utilise toute la bonté de vos clés ssh pour toutes les commandes, rendant la plupart des actions à distance « Just Work ™ »

Cependant, il y a des moments où cela peut être déroutant, par exemple, lorsque vous avez deux clés chargées dans ssh-agent qui correspondent à différents comptes github ou bitbucket. Quel compte vous obtenez sera un petit jeu amusant à découvrir ;-/

La stratégie très simple suivante vous permet de vérifier les clés ssh par projet de manière très naturelle, puis de faire en sorte que capistrano respecte ces clés pour toutes les opérations 'cap XXX'.

Tout d'abord, vous voudrez créer un répertoire .ssh/ dans votre projet, puis y déposer toutes les clés pertinentes. Vous pouvez avoir plusieurs paires, assurez-vous simplement qu'elles sont nommées en utilisant la convention *.id_rsa habituelle.

a:~/git/dojo4/rails_app_m $ ls .ssh
id_rsa.dojo4            id_rsa.dojo4.pub

Ensuite, vous voudrez quelque chose comme ceci dans votre Capfile, qui détecte ces clés locales, si elles sont présentes, et les utilise via un ssh-agent temporaire - sinon, revenez à vos clés ssh-agent chargées globalement.

# si ce projet a un répertoire .ssh/, utilisez ces clés pour le déploiement.
# sinon utilisez les clés globales. les clés locales sont utilisées en lançant
# un ssh-agent temporaire et en ajoutant ces clés locales.
#
# probablement vous auriez une clé (s) qui permet l'accès à un hôte et un dépôt particulier
# (github, bitbucket, etc) dans votre répertoire .ssh local.
#
# probablement vous voulez .gitignore .ssh/ !
#
  dot_ssh = File.expand_path("#{ rails_root }/.ssh", __FILE__)
  globs = [
    File.join(dot_ssh, "*id_rsa"),
    File.join(dot_ssh, "id_rsa*"),
  ]
  ssh_keys = [
    ENV['SSH_KEYS'], globs.map{|glob| Dir.glob(glob)}
  ].join(' ').scan(/[^\s]+/).delete_if{|ssh_key| ssh_key =~ /\.pub$/}
  unless ssh_keys.empty?
    require 'session'
    sh = Session::Sh.new
    at_exit{ sh.close }
    sh.execute 'eval `ssh-agent -s -t 3600`'
    ssh_auth_sock = sh.execute('echo $SSH_AUTH_SOCK').first.to_s.strip
    ssh_agent_pid = sh.execute('echo $SSH_AGENT_PID').first.to_s.strip
    ENV['SSH_AUTH_SOCK'] = ssh_auth_sock
    ENV['SSH_AGENT_PID'] = ssh_agent_pid
    ssh_keys.each do |ssh_key|
      if system "SSH_AUTH_SOCK=#{ ssh_auth_sock } ssh-add #{ ssh_key.inspect } >/dev/null 2>&1"
        Say.say("Utilisation de la clé ssh #{ ssh_key.inspect }", :color => :green)
      else
        Say
