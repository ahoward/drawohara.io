@drawohara

Comme beaucoup d'autres boutiques, nous gérons de nombreux petits problèmes. Des problèmes comme où obtenir notre café (Boxcar), où aller pour le déjeuner, quel éditeur utiliser (Vim).

Nous avons également lutté pour savoir comment gérer l'accès à nos serveurs de développement, de préproduction et de production. Nous avons essayé quelques itérations pour résoudre ce problème. Puis nous avons eu deux réalisations:

  • Tous ceux qui développent et déploient du code sont dans une seule équipe sur GitHub
  • GitHub fournit l'accès à la clé SSH publique pour leur utilisateur via leur API

Il s'avère que les clés SSH publiques sont en effet publiques. L'arrivée imminente de Miles Matthias a fourni la motivation pour agir sur cette intuition. lockpick est né de cette réalisation.

Nous avons maintenant le script lockpick en tâche cron. Tout membre de l'équipe dojo4 et des sous-équipes de confiance a désormais accès à nos serveurs.

lockpick est un logiciel open source et est sous licence MIT.
