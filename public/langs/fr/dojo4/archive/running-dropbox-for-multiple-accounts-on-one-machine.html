J'ai configuré une nouvelle machine cette semaine et j'avais vraiment besoin d'exécuter deux instances de Dropbox : une pour mes affaires personnelles et une pour mon compte http://dojo4.com.

Il existe de nombreuses façons de le faire, mais la méthode suivante est, de loin, la plus simple et la plus robuste.

Elle supporte également l'exécution de plus de 2 instances sur la même machine.

La première étape consiste à créer un script shell en utilisant votre éditeur de texte préféré (vim) et à le placer n'importe où. Il devrait ressembler à ceci :

#!/usr/bin/env bash
# ce fichier est enregistré à : /Users/$USER/bin/dojo4.dropbox
HOME=/Users/$USER/dojo4 /Applications/Dropbox.app/Contents/MacOS/Dropbox

Remplacez ‘dojo4’ ci-dessus par ce que vous voulez appeler votre autre instance. L'exécution directe de ce script démarrera Dropbox avec son répertoire ‘Dropbox’ normal imbriqué sous un dossier ‘dojo4’.

Cependant, vous n'allez pas exécuter cela directement. Vous allez télécharger Platypus ( http://sveinbjorn.org/platypus ).

Vous avez presque terminé. Téléchargez et installez-le. En raison des paramètres de sécurité, vous devrez peut-être autoriser les installations non-AppStore sur votre machine :

Ensuite, lancez Platypus et créez une nouvelle application de script shell

Dans mon cas, j'ai nommé ceci ‘Dojo4Dropbox’ et cette nouvelle application Dropbox enveloppée est enregistrée sous ‘/Applications/Dojo4Dropbox.app’.

C'est vraiment tout. Vous pouvez exécuter cette nouvelle application comme ‘/Applications/Dropbox.app’ et son répertoire de stockage sera imbriqué. Et vous pouvez la gérer comme un élément de démarrage comme n'importe quelle autre application :

Répétez l'opération pour exécuter plus de 2 applications Dropbox. ;-)
