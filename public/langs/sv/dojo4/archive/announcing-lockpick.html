Likt många andra företag, hanterar vi många mindre problem. Problemen som varifrån vi ska få vårt kaffe från (Boxcar), vart vi ska gå och äta lunch, vilken editor vi ska använda (Vim).

Vi hade även problem med hur vi skulle hantera vem som har tillgång till våra utvecklings-, test- och produktionsservrar. Vi gick igenom några iterationer för att lösa detta problem. Vi kom då fram till två insikter:

  • Alla som utvecklar och distribuerar kod är i ett enda team på GitHub
  • GitHub tillhandahåller tillgång till användarens offentliga SSH-nyckel via deras API

Det visade sig att SSH offentliga nycklar verkligen är offentliga. Den förestående ankomsten av Miles Matthias gav motivationen att agera på denna insikt. lockpick kom ur den insikten.

Vi har nu lockpick-skriptet på ett cron-jobb. Varje medlem av dojo4 och betrodda underteam har nu tillgång till våra servrar.

lockpick är öppen källkodsprogramvara och licensieras under MIT-licensen.
