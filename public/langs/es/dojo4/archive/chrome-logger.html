Una herramienta genial llamada Chrome Logger ha estado disponible por un tiempo, lo que permite, a través de una especificación conocida, el registro del lado del cliente desde el servidor. Parece haber sido completamente reescrita recientemente, pero solo tenía implementaciones en PHP y Python.

Después de un rápido trabajo anoche, pude publicar una versión temprana de una implementación en Ruby. Es un middleware de Rack empaquetado como una gema que expone env['console'] desde el cual puedes crear mensajes de registro.

También viene con una fácil integración de Rails: require 'chrome_logger/railtie'

Puedes ver un ejemplo, leer el código, o simplemente ¡instalarlo!

gem install chrome_logger
