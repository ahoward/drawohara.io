Como muchas otras tiendas, lidiamos con muchos problemas pequeños. Problemas como de dónde obtener nuestro café (Boxcar), a dónde ir a almorzar, qué editor usar (Vim).

También luchamos con cómo gestionar quién tiene acceso a nuestros servidores de desarrollo, pruebas y producción. Pasamos por algunas iteraciones intentando resolver este problema. Luego tuvimos dos realizaciones:

  • Todos los que desarrollan y despliegan código están en un solo equipo en GitHub
  • GitHub proporciona acceso a la clave SSH pública para su usuario a través de su API

Resulta que las claves SSH públicas son realmente públicas. La inminente llegada de Miles Matthias proporcionó la motivación para actuar sobre esta idea. lockpick surgió de esa realización.

Ahora tenemos el script lockpick en un trabajo cron. Cualquier miembro del equipo dojo4 y subequipos de confianza ahora tiene acceso a nuestros servidores.

lockpick es software de código abierto y está licenciado bajo la licencia MIT.
