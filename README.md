# TL;DR;

### 1) ensure you have the correct version of ruby setup, prefer `rbenv`
```sh
  # these should be the same!
  ~> cat .ruby-version
  ~> ruby --version
```

### 2) ensure you have the correct version of node setup, prefer `nvm`
```sh
  # these should be the same!
  ~> cat .node-version
  ~> node --version
```
