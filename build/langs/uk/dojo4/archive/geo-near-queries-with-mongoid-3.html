сьогодні я розгортав новий функціонал пошуку і дійсно потребував функціоналу ‘geoNear’ від mongo

http://docs.mongodb.org/manual/reference/command/geoNear/

на жаль, гем mongoid (3) не підтримує це безпосередньо.

на щастя, код durran’а (https://twitter.com/modetojoy) досить якісний, і кілька хвилин пошуку дали рішення, яке призвело до наступного коду:

# http://stackoverflow.com/questions/5319988/how-is-maxdistance-measured-in-mongodb
# 1° широти = 69.047 миль = 111.12 кілометрів
#
  GRADUSI_NA_MILYU = 1 / 69.047
  MILI_NA_GRADUS = 1 / GRADUSI_NA_MILYU
  def Store.znajdi_vsix_za_shyrotoju_i_dovhotoju(shyrota, dovhota, opcii = {})
    opcii.do_opcii!
    mili = opcii[:mili] || 100
    maks_vidstal = mili * GRADUSI_NA_MILYU
    doc = 
      Mongoid.sesiya(:z_zagalnomu).
        komanda(
          :geoNear     => Store.nazva_kollekcii,
          :near        => {:shyrota => shyrota, :dovhota => dovhota},
          :maxDistance => maks_vidstal
        )
    magazini = []
    if doc['ok'] == 1.0
      rezultati = Array(doc['rezultati'])
      rezultati.dlia_kozhnoho do |rezultat|
        vidstal_u_gradusah = rezultat['vid']
        obj = rezultat['obj']
        magazin = Store.vstanov(obj)
        magazin['vidstal'] = Float(vidstal_u_gradusah) * MILI_NA_GRADUS
        magazini.dodai(magazin)
      end
    end
    magazini
  end
