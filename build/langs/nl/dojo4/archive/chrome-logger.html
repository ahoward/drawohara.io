/chrome-logger
gepubliceerd op: 2013-04-10

Er is al een tijdje een leuk gereedschap genaamd Chrome Logger dat, via een bekende specificatie, client-side logging vanaf de server toelaat. Het lijkt recent een volledige herschrijving te hebben ondergaan maar had alleen PHP en Python implementaties.

Na wat snel werk gisteravond, kon ik een vroege versie van een Ruby implementatie uitbrengen. Het is een Rack middleware verpakt als een gem die env['console'] blootstelt waar je logberichten mee kunt maken.

Het komt ook met gemakkelijke Rails integratie: require 'chrome_logger/railtie'

Je kunt een voorbeeld bekijken, de code lezen, of het gewoon installeren!

gem install chrome_logger
