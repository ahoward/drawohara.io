Vervang 'dojo4' boven door wat je ook wilt noemen voor je alternatieve instantie. Het uitvoeren van dit script start Dropbox met de normale 'Dropbox'-map genest onder een 'dojo4'-map.

Je gaat dit echter niet rechtstreeks uitvoeren. Je gaat Platypus ( http://sveinbjorn.org/platypus ) downloaden.
