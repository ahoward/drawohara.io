les applications web ont presque toujours besoin d'une pléthore de petites méthodes d'aide javascript : vous savez, les formulaires à complétion automatique, le remplissage des valeurs par défaut, les validations, etc. Ce ne sont pas des appels API proprement dits, juste de petits bouts de fonctionnalité nécessaires pour faire fonctionner les vues.

il y a toujours une question de savoir à quel contrôleur rattacher ces méthodes. Doit-on créer un contrôleur d'aide global pour toutes ces choses ? Doit-on les rattacher au contrôleur en question ? Comment partager le backend des méthodes d'aide javascript entre les contrôleurs ?

étape un

inclure le module RPC dans votre ApplicationController

  class ApplicationController < ::ActionController::Base
    include RPC
  end

étape deux

dans les contrôleurs, déclarer les méthodes d'aide rpc

  class PostsController < ApplicationController
    rpc(:defaults) do |params|
      post = Post.new(params)
      post.set_defaults!
      post.as_json
    end
  end
