```html Оголошення Lockpick
@drawohara

Як і багато інших компаній, ми маємо справу з багатьма дрібними проблемами. Проблеми типу де замовляти каву (Boxcar), куди ходити на обід, який редактор використовувати (Vim).

Ми також боролися з тим, як керувати доступом до наших серверів розробки, тестування та виробництва. Ми пройшли кілька ітерацій, намагаючись вирішити цю проблему. Тоді ми зробили два важливі висновки:

  • Всі, хто розробляє та розгортає код, знаходяться в одній команді на GitHub
  • GitHub надає доступ до публічного SSH ключа для свого користувача через їх API

Виявилося, що SSH публічні ключі дійсно публічні. Наближення прибуття Майлза Меттіаса надало мотивацію діяти згідно цього усвідомлення. lockpick виник з цього усвідомлення.

Тепер у нас є скрипт lockpick на cron завданні. Будь-який член команди dojo4 та довірених підкоманд має доступ до наших серверів.

lockpick є програмним забезпеченням з відкритим вихідним кодом і ліцензується за ліцензією MIT.
```
