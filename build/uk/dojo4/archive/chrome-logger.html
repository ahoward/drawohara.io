Інструмент під назвою Chrome Logger існує вже давно та дозволяє, завдяки відомій специфікації, клієнтське логування з боку сервера. Здається, він нещодавно зазнав повного переписування, але були лише PHP та Python імплементації.

Після швидкої роботи минулої ночі, я зміг випустити ранню версію Ruby імплементації. Це Rack middleware, запакований як gem, який надає доступ до env['console'], з якого можна створювати повідомлення логу.

Він також має легку інтеграцію з Rails: require 'chrome_logger/railtie'

Ви можете переглянути приклад, прочитати код, або просто встановити його!

gem install chrome_logger
