rails, inclusief rails3, heeft deze prachtige stukje code gerelateerd aan het bieden van een "to_json" methode op ingebouwde functies:

# Hack om de json gem eerst te laden zodat we de to_json methode ervan kunnen overschrijven.
    begin
      require 'json'
    rescue LoadError
    end
    # De JSON gem voegt enkele modules toe aan Ruby core klassen die de definitie van :to_json bevatten, die het standaardgedrag overschrijven. Dit gezegd, moeten we de basis to_json methode defini√´ren in alle van hen,
    # anders zullen ze altijd de to_json gem implementatie gebruiken, die in meerdere gevallen achterwaarts incompatibel is
    # (bijvoorbeeld, de JSON implementatie voor Hash werkt niet met erfenis en gevolgelijk kunnen klassen zoals ActiveSupport::OrderedHash niet naar json worden geserialiseerd.
    [Object, Array, FalseClass, Float, Hash, Integer, NilClass, String, TrueClass].each do |klass|
      klass.class_eval <<-RUBY, __FILE__, __LINE__
# Dumps object in JSON (JavaScript Object Notation). See www.json.org for more info.
def to_json(options = nil)
ActiveSupport::JSON.encode(self, options)
end
RUBY
    end

wat dit effectief doet, is de "to_json" methode die de json gem biedt, te overschrijven met een die active_support beter vindt. De opmerking dat json kapot is, is verouderd en dit was een hack om te beginnen met. In mijn geval vind ik dit soort dingen echt leuk in mijn /api controller:

  def to_json(object)
    if Rails.env.production?
      JSON.generate(object)
    else
      JSON.pretty_generate(object)
    end
  end

omdat het debuggen een stuk eenvoudiger maakt. Hier is hoe ik het deed in een rails3 app -

De eerste stap was om rails3 te laten ondersteunen het "pre-initialiseren" concept dat oude rails gebruikte:

    ### file: config/preinitializer.rb
    dirname = File.dirname(File.expand_path(__FILE__))
    glob = File.join(dirname, 'preinitializers', '**/**.rb')
    preinitializers = Dir.glob(glob)
    preinitializers.each{|preinitializer| Kernel.load(preinitializer)}

Daarna voegde ik deze pre-initialisator toe:

### file: config/preinitializers/un_fuck_to_json.rb
    begin
      require 'rubygems'
    rescue LoadError
    end
    begin
      require 'json'
    rescue LoadError
    end
    [Object, Array, FalseClass, Float, Hash, Integer, NilClass, String, TrueClass].each do |klass|
      klass.class_eval do
        alias_method('__to_json__', 'to_json') if method_defined?('to_json')
      end
    end
