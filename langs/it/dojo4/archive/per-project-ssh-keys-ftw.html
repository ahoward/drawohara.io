Con le distribuzioni Rails + Capistrano questa impostazione

ssh_options[:forward_agent] = true

è praticamente l'impostazione predefinita e sana. Utilizza tutta la bontà delle tue chiavi ssh per tutti i comandi, rendendo la maggior parte delle azioni remote 'Funziona e basta ™'

Tuttavia, ci sono momenti in cui questo può essere confuso, ad esempio quando hai due chiavi caricate in ssh-agent che mappano account github o bitbucket diversi. Quale account ottieni sarà un piccolo gioco divertente da capire ;-/

La seguente strategia molto semplice ti consente di inserire chiavi ssh per progetto in modo molto naturale e di far sì che capistrano rispetti queste chiavi per tutte le operazioni 'cap XXX'.

Innanzitutto, vorrai creare una directory .ssh/ nel tuo progetto, quindi scarica eventuali chiavi rilevanti lì. Puoi avere molte coppie, assicurati solo che siano denominate utilizzando la convenzione *.id_rsa.

a:~/git/dojo4/rails_app_m $ ls .ssh
id_rsa.dojo4            id_rsa.dojo4.pub

Successivamente, vorrai qualcosa del genere nel tuo Capfile, che rileva queste chiavi locali, se presenti, e le utilizza tramite un agente ssh-agent temporaneo - altrimenti ripiega sulle chiavi caricate nel tuo agente ssh-agent globale.

# se questo progetto ha una directory .ssh/, utilizza quelle chiavi per la distribuzione.
# altrimenti utilizza quelle globali. le chiavi locali vengono utilizzate avviando
# un agente ssh temporaneo e aggiungendo quelle chiavi locali.
#
# probabilmente avresti una chiave (o più chiavi) che consentono l'accesso a un particolare host e repo
# (github, bitbucket, ecc.) nella tua directory .ssh/ locale.
#
# probabilmente vuoi .gitignore .ssh/ !
#
  dot_ssh = File.expand_path("#{ rails_root }/.ssh", __FILE__)
  globs = [
    File.join(dot_ssh, "*id_rsa"),
    File.join(dot_ssh, "id_rsa*"),
  ]
  ssh_keys = [
    ENV['SSH_KEYS'], globs.map{|glob| Dir.glob(glob)}
  ].join(' ').scan(/[^\s]+/).delete_if{|ssh_key| ssh_key =~ /\.pub$/}
  unless ssh_keys.empty?
    require 'session'
    sh = Session::Sh.new
    at_exit{ sh.close }
    sh.execute 'eval `ssh-agent -s -t 3600`'
    ssh_auth_sock = sh.execute('echo $SSH_AUTH_SOCK').first.to_s.strip
    ssh_agent_pid = sh.execute('echo $SSH_AGENT_PID').first.to_s.strip
    ENV['SSH_AUTH_SOCK'] = ssh_auth_sock
    ENV['SSH_AGENT_PID'] = ssh_agent_pid
    ssh_keys.each do |ssh_key|
      if system "SSH_AUTH_SOCK=#{ ssh_auth_sock } ssh-add #{ ssh_key.inspect } >/dev/null 2>&1"
        Say.say("Utilizzo della chiave ssh #{ ssh_key.inspect }", :color => :green)
      else
