/chrome-logger
publicado en: 2013-04-10

Una herramienta genial llamada Chrome Logger ha existido por un tiempo que permite, a través de una especificación conocida, el registro de clientes desde el servidor. Parece haber sufrido una reescritura completa recientemente, pero solo tenía implementaciones en PHP y Python.

Después de un rápido trabajo anoche, pude lanzar una versión temprana de una implementación en Ruby. Es un middleware de Rack empaquetado como una gema que expone env['console'] desde donde puedes crear mensajes de registro.

También viene con una fácil integración de Rails: require 'chrome_logger/railtie'

Puedes ver un ejemplo, leer el código, o simplemente ¡instalarlo!

gem install chrome_logger
