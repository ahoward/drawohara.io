Acabamos de lanzar un sitio web de marketing para Planted Nutrition, una nueva práctica de nutrición en Boulder dirigida por Ellen Teig. 

Planted Nutrition es un negocio nuevo, que apenas comienza a cocinar, y Ellen quería un sitio web brillante y profesional para ayudar a los clientes a aprender más sobre sus servicios y enfoque. El presupuesto y las necesidades de Ellen eran pequeños, por lo que decidimos crear un sitio web de una página rápido que mostrara todo su contenido.

Las inspiraciones de diseño de Ellen incluyeron varios blogs culinarios con grandes y hermosas imágenes de comida. Anthony Dimitre llevó ese aspecto audaz a la página con un par de zanahorias muy bonitas.

Anthony también creó un logotipo muy bonito que dará sabor a todos los demás materiales de Planted Nutrition: tarjetas de visita, paquetes de clientes y más.

Detrás de escena, Planted Nutrition es un sitio web estático construido con Middleman. No podemos decir lo suficiente sobre esta herramienta. La generación de sitios estáticos no es nueva y Middleman no es el primer generador de sitios estáticos, pero es una herramienta muy buena.

En dojo4 estamos usando Middleman como alternativa a los sistemas de gestión de contenido LAMP omnipresentes. Hay varias buenas herramientas de CMS LAMP, pero no son de talla única y todas requieren algunos compromisos. En este caso, incluso un CMS tan simple como WordPress sería demasiado complejo y requeriría mucho mantenimiento para las necesidades inmediatas de Planted Nutrition.

Algún día, Planted Nutrition puede querer algo más que un sitio estático. Mientras tanto, este sitio delicioso es suficiente.
